# Roadmap

## Stage 1: Documentation

- Document the existing Google Sheets prototype logic.
- Define key social media metrics.
- Define the goal-tracking structure.
- Create example datasets.
- Describe common reporting problems for marketers.

## Stage 2: First working tools

- Build simple scripts for cleaning exported data.
- Build basic plan vs. fact tracking.
- Create examples for CSV, Excel, and Google Sheets workflows.
- Add first UTM and campaign check templates.

## Stage 3: Public examples

- Add beginner-friendly documentation.
- Add examples in English and Russian.
- Create tutorials for non-technical marketers.
- Invite feedback from marketers and SMM specialists.

## Stage 4: Extended Marketing Ops Toolkit

- Add reporting automation examples.
- Add CRM/export workflow examples.
- Add budget plan vs. fact templates.
- Explore safe integrations with official APIs where available.
