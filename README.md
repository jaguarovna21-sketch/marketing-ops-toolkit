# Marketing Ops Toolkit

An open-source toolkit for marketers, SMM specialists, and small teams who need practical automation but often do not have access to developers, APIs, or expensive BI systems.

## Why this project exists

Many marketers still collect data manually from different social media dashboards, CRMs, ad platforms, spreadsheets, screenshots, and reports.

This creates several problems:

- too much manual work;
- human errors in reporting;
- unclear goals and progress tracking;
- different metrics across different teams;
- lack of access to APIs or engineering support;
- less time for strategy, creativity, and business growth.

This project aims to make practical marketing automation more accessible to non-technical specialists.

## First module: Social Media Analytics & Goal Tracker

The first planned module is a Social Media Analytics & Goal Tracker.

It is based on a Google Sheets prototype that I created and tested on a real marketing project. During one year of using this system, the project’s social media metrics grew by more than 1000%.

The system helped track goals, compare progress, understand what was working, and make decisions based on data. The main limitation of the current version is that it requires too much manual work.

The goal of this module is to automate repetitive parts of the workflow and make the approach reusable for other marketers and SMM specialists.

## Planned features

- social media analytics;
- goal tracking;
- plan vs. fact reporting;
- marketing report automation;
- UTM and campaign checks;
- CSV, Excel, and Google Sheets workflows;
- beginner-friendly documentation for non-technical users;
- examples in English and Russian.

## Project status

This project is currently in an early planning/prototype stage.

The first version will focus on documenting the existing Google Sheets logic, creating example datasets, and building simple tools for marketers who work with exported data and spreadsheets.

## Who this is for

This project is for:

- marketers;
- SMM specialists;
- small business owners;
- marketing teams without developers;
- people who want to automate routine reporting without becoming full-time software engineers.
